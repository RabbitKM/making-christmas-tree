# 客製化聖誕樹小遊戲

一個簡單的網頁小遊戲，讓您可以客製化自己的聖誕樹！

## 功能
- 選擇樹的高度 (3-10 層)
- 選擇樹的形狀：三角形、星星形、松樹形
- 選擇樹的顏色：綠色、藍色、紅色
- 添加裝飾：星星、燈泡、裝飾球

## 如何使用
1. 打開 `index.html` 在瀏覽器中。
2. 選擇您的偏好設定。
3. 點擊 "繪製聖誕樹" 按鈕。
4. 享受您的客製化聖誕樹！

## 發佈到 GitHub Pages
1. 將此專案推送到 GitHub 倉庫。
2. 在倉庫設定中，啟用 GitHub Pages。
3. 選擇 "main" 分支作為來源。
4. 您的遊戲將可在 `https://your-username.github.io/your-repo-name/` 存取。

## 技術
- HTML5
- CSS3
- JavaScript (ES6+)
